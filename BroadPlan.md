ODIN: Multi-agent code review system with Graph RAG and MCP integration. Build specialized LangGraph agents (security reviewer, performance analyzer, documentation validator) that collaborate through a graph-based knowledge base of codebase patterns. Expose it as an MCP server so it plugs into Claude Code, Cursor, and VS Code. Deploy on GCP with Cloud Run and instrument with Langfuse. This single project demonstrates every trending capability in one production-ready tool.
